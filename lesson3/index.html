Arctic Blast Forecast http://www.articblastforcast.com

Site Purpose

This site is designed to allow the user to check on the current weather for their town The page will show an image reflecting the type of weather for the day. The page will also show warnings for any hazardous weather.

The full page site will also display a 5-day forcast.
