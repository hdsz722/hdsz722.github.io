Site Name

Arctic Blast

Site URL

http://www.articblastweather.net

Site Purpose

This site is designed to allow the user to check on the current weather for their town. For a quick reference, the page will also display an image reflecting the type of weather for the day. The page will also display any forcasted warnings for hazardous weather to help users prepare and stay safe.

This website will be responsive so users will be able to access the weather from mulitple devices. The mobile display will give a summary of the weather for their town, plus a paragraph giving the details of the weather for that area and any warnings. The medium sized displays will provide the user with a stacked 5day forcast for the weather in their area. The full page site will also display a 5-day forcast banner.
